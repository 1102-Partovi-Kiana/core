Project Description

Our project CORE (Center for Optimizing Robotics Education) is an educational web platform for undergraduate students wanting to grasp the fundamentals of robotics with ease. Our project is intended to provide undergraduate students with a virtual robotics lab. They will learn about how to program a robot in a live environment after going through course content. The platform includes robotics courses, robotic coding environments, and teacher and student interaction. The robotic environment is created from MuJoCo, an open-source physics simulation engine. The coding environment is in python and the code entered in this environment will affect the robot simulation. Our platform is designed to offer a comprehensive educational experience where the user has access to a dashboard specific to their role. Through this portal, instructors can see our course content, manage grades, and monitor students. On the other hand, students can access the learning material which contains a virtual environment to simulate hands-on learning with gamification features to promote engagement from the student.
