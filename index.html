Project Description

Our project CORE (Center for Optimizing Robotics Education) is an educational web platform for undergraduate students wanting to grasp the fundamentals of robotics. Our project is intended to provide undergraduate students with a virtual robotics lab. They will be able to learn about how to code a robot in python, and apply the content they have learned through taking quizzes and coding in a live environment. Our platform includes robotics courses, robotic environments, coding environments, teacher and student interaction, and gamification. Students will learn about a specific robot and the functions that go along with it. For example, this can include how to move the robot or move objects/assets related to the robot with python code. The robotic environment is created from MuJoCo, an open source physics simulation engine. The coding environment is in python and the code entered in this environment will affect the robot simulation. Our goal is to provide for all types of students, whether that be because they donâ€™t have access to purchase hardware equipment, have access to a lab, or are intimidated by robotics. We aspire to make robotics an engaging and accessible experience for all. Our platform is designed to offer a comprehensive educational experience where the user has access to a dashboard specific to their role. Through this portal, instructors can see our course content, manage grades, and monitor students. On the other hand, students can access the learning material which contains a virtual environment to simulate hands-on learning with gamification features to promote engagement from the student. Additionally, there will be an AI-powered chatbot that is integrated to provide support depending on the page or to provide hints to students as they learn. Regarding the frontend, technologies we are using include HTML, CSS, JavaScript, Bootstrap, Jinja2, and CodeMirror. In the backend we are using Python, Flask, PostgresQL, and the Gemini API. Our Robotics are powered through OpenAI Gym, and MuJoCo.
